# 予約くん

登録不要・URLを発行するだけですぐに予約受付を開始できる、シンプルで軽量な予約システムです。

PHPとBootstrap 5で構築されており、個人レッスン、面談予約、イベント受付、相談会予約など幅広い用途で利用できます。

---

## 主な機能

* 登録不要ですぐ利用開始
* 予約ページをURLで共有可能
* スマートフォン対応
* 日本語・英語対応
* Bootstrap 5によるレスポンシブデザイン
* 予約枠の追加・削除
* 予約人数制限
* 予約者による自己キャンセル機能
* 管理者による予約キャンセル機能
* キャンセルURLの自動発行
* メールアドレス入力（任意）

## インストール方法

1. リポジトリをダウンロードまたはクローンします。

```bash
git clone https://github.com/s0323861/yoyakukun.git
```

2. Webサーバー上に配置します。

3. データ保存用ディレクトリに書き込み権限を付与します。

4. ブラウザで `index.php` にアクセスします。

---

## 使い方

### 1. 予約ページを作成

管理画面から日時や定員を設定して予約ページを作成します。

### 2. URLを共有

生成されたURLを参加者に共有します。

### 3. 予約受付

利用者は名前とメールアドレス（任意）を入力して予約します。

### 4. キャンセル対応

* 利用者はキャンセルURLから予約を取り消せます。
* 管理者は管理画面から予約を削除できます。
